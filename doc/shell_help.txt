--Help--

Documentation format:
	{description}: '{command char}' ([arguments])

	{description}:  description of the command
	{command char}: char used to invoke the command
	[arguments]:    (optional) argument format string.

	e.g.
		save file: 's' \"%s %s\"
		's' saves a file, and requires two string arguments.

add line: 'l' (%d %d %d %d %d %d)

create scale transformation: 's' (%d %d %d)
create translation: 't' (%d %d %d)
create rotation through x-axis: 'x' (%d)
create rotation through y-axis: 'y' (%d)
create rotation through z-axis: 'z' (%d)
draw points to screen: 'v'
draw points to screen, and save to file: 'g' (%s)
